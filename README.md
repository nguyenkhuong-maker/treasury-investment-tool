# Treasury Master Dashboard

Folder chính duy nhất để anh làm việc sau này.

## File đang dùng chính
- `index.html`
  - lấy từ bản hoàn chỉnh ở `treasury-rate-dashboard/index.html`
  - đây là file dashboard lãi suất / đầu tư anh đang dùng
- `company-loans.html`
  - bản HTML riêng cho phần dư nợ vay công ty để tham chiếu và phát triển tiếp
- `data/history.json`
  - dữ liệu lãi suất theo tháng / tuần cho `index.html`
- `data/company-loans-current.json`
  - snapshot dư nợ vay công ty hiện tại, không lưu history

## File hỗ trợ
- `SCHEMA.md`
  - mô tả schema chuẩn cho dữ liệu lãi suất và dư nợ vay
- `assets/company-loan/`
  - ảnh/snapshot hỗ trợ từ phần company loan nếu sau này cần đối chiếu
- `archive/`
  - lưu bản scaffold cũ của `treasury-master-dashboard`

## Ghi chú vận hành
- Từ giờ nếu làm tiếp, ưu tiên làm trong folder này:
  - `treasury-master-dashboard/`
- Hai folder cũ `treasury-rate-dashboard/` và `company-loan-dashboard/` có thể xem là nguồn tham chiếu / backup.
- Khi anh chốt xong hoàn toàn, mình có thể dọn tiếp 2 folder cũ sau.
